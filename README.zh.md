# reggin

`reggin` 是一个轻量级的包，用于以结构化和可重用的方式注册和管理 Gin HTTP 路由。

## 特性
- 使用自定义响应类型定义应用路由。
- 使用 Gin 的 `SecureJSON` 对 JSON 响应进行安全处理。

## 安装

```bash
go get github.com/yyle88/reggin
```
